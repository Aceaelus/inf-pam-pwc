Czym jest Firebase?

Firebase to platforma oferująca narzędzia do budowania oraz optymalizowania aplikacji. Podstawową funkcjonalnością Firebase jest tworzenie baz danych oraz usługa Backend as a Service (BasS).

Systemy, w których można zaimplementować Firebase

  • Android
  • iOS
  • Unity
  • Web

Implementacja Firebase

  1. Rejestracja aplikacji w konsoli Firebase
  2. Dodanie pliku konfiguracyjnego do projektu

Narzędzia Firebase

Cloud Firestore

Baza danych typu NoSQL, której struktura oparta jest na kolekcjach i dokumentach.


							val person = hashMapOf(
								"forename" to "Adam",
								"surname" to "Nowak",
								"born" to 1990,
								"married" to false
							)
							db.collection("users").document("anowak")
								.set(person)
								.addOnSuccessListener { Log.d(TAG, "DocumentSnapshot successfully written!") }
        						.addOnFailureListener { e -> Log.w(TAG, "Error writing document", e) }

Realtime Database

Baza danych typu NoSQL, w której możemy przechowywać i synchronizować dane między naszymi użytkownikami w czasie rzeczywistym.

Authentication

Firebase Authentication jest narzędziem dostarczającym nam usługi przeznaczone do uwierzytelniania użytkowników.

Najczęściej wykorzystywane metody uwierzytelniania:

  • Email + password
  • Google
  • Facebook

							FirebaseAuth.getInstance().createUserWithEmailAndPassword(email, password)
            				.addOnCompleteListener {
               					if (!it.isSuccessful) return@addOnCompleteListener}
            				.addOnFailureListener {
                				Toast.makeText(this, "${it.message}", Toast.LENGTH_LONG).show()}

Storage

Firebase Storage pozwala nam na przechowywanie danych mulitmedialnych.


							val filename = UUID.randomUUID().toString()
    			val ref = FirebaseStorage.getInstance().getReference("/images/$filename")

							ref.putFile(selectedPhotoUri!!)

Hosting

Firebase Hosting pozwala nam na szybkie wdrożenie naszej aplikacji.

Crashlytics

Narzędzie raportujące awarie w aplikacji w czasie rzeczywistym, które pochodzą z urządzeń użytkowników na których zainstalowana jest aplikacja.

Test Lab

Oparta na chmurze infrastruktura do testowania aplikacji, która umożliwia testowanie aplikacji na różnych urządzeniach i konfiguracjach

Google Analytics

Zapewnia nam wgląd w wykorzystywanie naszej aplikacji oraz zaanagażowanie jej użytkowników.
